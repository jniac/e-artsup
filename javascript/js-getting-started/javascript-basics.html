Les bases du Javascript

Les principaux "types" :

String

String = chaîne de caractères

Les chaînes de caractères permettent de représenter du texte.

let a = "hello" let b = "world" alert(a + " " + b)

Number

Les "Number" permettent de représenter un très grand nombre de nombres (sic).

let c = 10 let d = .25 alert(c + d * 3)

Attention toutefois, les valeurs possibles sont limitées, ainsi des erreurs de calculs sont possibles :

alert(.1 + .2) alert("étonnant n'est-ce pas ?")

Array

"Array" = tableau (d'éléments)

let fruits = ["banana", "orange", "kiwi"] alert("il y a " + fruits.length + " fruits") alert("le premier fruit est " + fruits[0]) alert("le dernier fruit est " + fruits[fruits.length - 1])

Les "boucles"

For loop

Un exemple classique, une boucle sur i

for (let i = 0; i < 10; i++) { console.log(i) }

Boucle sur un tableau de valeur (à l'ancienne):

let items = [1, 10, 1000, "etc!"] for (let i = 0; i < items.length; i++) { console.log(i, items[i]) }

Boucle sur un tableau de valeur nouvelle façon (stylé) :

let items2 = [1, 10, 1000, "etc!"] for (let value of items2) { console.log(value) }
